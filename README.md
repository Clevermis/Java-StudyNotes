###### 开发前的配置要求

1. JDK 1.8
2. Mysql 8.0
3. Maven 3.6.3
4. Tomcat 9.0.7

###### **安装步骤**


1. Clone the repo

```sh
git clone https://github.com/Clevermis/java-study.git
```
